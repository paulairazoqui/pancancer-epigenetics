# data

Data directories follow roadmap v3.0.

- `raw/`: immutable source datasets, organized by source dataset.
- `interim/`: harmonized and analysis-ready inputs for downstream notebooks.
- `processed/`: stable biological entities and analytical outputs generated by the framework.

Raw files should remain read-only after download. Any derived or harmonized file should be written to `interim/` or `processed/` according to the roadmap.
